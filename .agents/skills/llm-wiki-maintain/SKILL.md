# LLM Wiki Maintain

Use this skill for routine upkeep.

## Steps

1. Rebuild generated indexes and catalogs after note changes.
2. Update the source manifest after new coverage is added.
3. Review uncovered Raw sources.
4. Add a log note for meaningful changes.
5. Run the maintenance gate before handoff or commit.
